# Доработки:

1. Дополнить таблицу Excel: Регион показа, ср. позиция показа, ср позиция клика, процент охвата.
2. Список площадок для минусовки давать в txt файле, последовательно для каждой кампании, в этом же файле писать сводку, как улучшить кампании.
3. В файле анализа xlsx вкладку Кампания убрать, оставить Кампании, ато они дублируют друг друга.
4. Файл анализа xlsx Вкладка "Устройство" в первый столбец добавить название кампании и по каждой кампании сделать отдельно аналитику по типу устройств.
5. Файл анализа xlsx Вкладки "Пол" и "Возрат" объединить на одну вкладку "Пол-возраст", т.к. их нужно анализировать только вместе Пол+Возраст и на одном листе Excel. В таблице сделать отдельно аналитику по Мужской + Возраст, Женский + возраст, не определен - по всем возрастам суммарно (это исключение и их минимум).
5. Файл анализа xlsx Вкладка "Тип условия показа" в первый столбец добавить название кампании и по каждой кампании сделать отдельно аналитику по типу условия показа.
6. Файл анализа xlsx Вкладки "Площадка", "Группа", "Заголовок", "Изображение" - на каждой из этих вкладок в первый столбец добавить название кампании и по каждой кампании сделать статистику отдельно. То есть на каждой из этих вкладок одна таблица, сортировка сначала по названию кампании, затем по столбцу spend по убыванию.
7. Столбец "spend" во всех таблицах переименовать на Расходы, руб с НДС.
8. Сделать вкладку по типу кампании, она должна быть второй по порядку в отчете. Типы кампании брать из названия кампании. Типы могут быть: поиск, рся, товарн, смарт-баннеры, динамическая. Для определения типа кампании название нужно привести к нижнему регистру, условие "содержит". Если в названии есть слово "динамическ", то это наиболее важный параметр для присвоения типа кампании, т.к. Динамическая часто называется Поиск Динамическая, и в таком случае, её нужо относить к Дианмическая.
