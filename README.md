# Анализатор отчетов Яндекс.Директ

> Разработан совместно с Claude Sonnet 4.5

Скрипт на Python для автоматического анализа Excel-отчётов из Яндекс.Директ: считает ключевые метрики (CPC, CPA, CR, CTR), строит сводки по кампаниям и срезам, формирует Excel-файл с отчётами и TXT-файл с рекомендациями.

---

## Быстрый запуск

1. **Настройте скрипт** — откройте `yandex_direct_analyzer.py`, найдите блок настроек в начале файла:
   ```python
   # ==================== НАСТРОЙКИ ====================
   REPORT_TITLE = "Отчет по проекту ОПОРА за период 1 - 31 марта 2026г."
   SITE_NAME    = "opora"   # используется для поиска файлов в папке data/
   DATA_FOLDER  = "data"
   OUTPUT_FOLDER = "out"
   MIN_SPEND    = 350.0     # минимальный расход для рекомендаций
   MIN_CLICKS   = 10        # минимум кликов для рекомендаций
   # ===================================================
   ```

2. **Подготовьте файлы данных** — положите Excel-файлы в папку `data/`:
   | Файл | Обязательный | Описание |
   |------|:---:|----------|
   | `mainstat_{SITE_NAME}*.xlsx` | ✅ | Основная статистика по структуре кампании |
   | `regions_{SITE_NAME}*.xlsx` | ➖ | Статистика по регионам |
   | `polvozrast_{SITE_NAME}*.xlsx` | ➖ | Статистика по полу и возрасту |
   | `pfrases_{SITE_NAME}*.xlsx` | ➖ | Статистика по поисковым фразам |

   Скрипт автоматически находит файлы по первым двум словам имени (`тип_сайт`).  
   Если опциональный файл не найден — соответствующая вкладка в отчёте не создаётся.

3. **Запустите скрипт:**
   ```powershell
   python yandex_direct_analyzer.py
   ```

4. **Получите результаты** в папке `out/`:
   - `{SITE_NAME}_analysis_result.xlsx` — Excel-отчёт
   - `{SITE_NAME}_recommendations.txt` — рекомендации и площадки для минусовки

## Структура Excel-отчёта

Вкладки формируются в следующем порядке:

| № | Вкладка | Описание |
|---|---------|----------|
| 1 | **Обзор** | Карточки с ключевыми метриками (расход, клики, конверсии, CPC, CPA, CR, CTR) |
| 2 | **Кампании** | Сводка по всем кампаниям |
| 3 | **Тип кампании** | Агрегация по типам кампаний |
| 4 | **Рекомендации** | Рекомендации и статус по каждой кампании |
| 5 | **Устройство** | Разбивка по устройствам в разрезе кампаний |
| 6 | **Устройства-все-камп** | Разбивка по устройствам суммарно по всем кампаниям |
| 7 | **Пол и возраст** | Разбивка по полу+возрасту в разрезе кампаний |
| 8 | **Пол и возраст-все-камп** | Разбивка по полу+возрасту суммарно по всем кампаниям |
| 9 | **Условие показа** | Разбивка по типу условия показа в разрезе кампаний |
| 10 | **Площадка** | Разбивка по площадкам в разрезе кампаний |
| 11 | **Группа объявлений** | Разбивка по группам в разрезе кампаний |
| 12 | **Заголовок** | Разбивка по заголовкам в разрезе кампаний |
| 13 | **Изображение** | Разбивка по изображениям в разрезе кампаний |
| 14 | **Регионы** | Из файла regions (если есть) |
| 15 | **Поисковые фразы** | Из файла pfrases (если есть) |

На каждой вкладке:
- Ячейка A1 — заголовок отчёта (шрифт 18, жирный)
- Ячейка A2 — название вкладки (шрифт 16, жирный)
- Таблица с данными начиная со строки 4
- Условное форматирование: цветовая шкала по расходам и CPA

## Форматирование числовых показателей

| Показатель | Формат |
|------------|--------|
| Расходы, руб с НДС | целое число, ₽ |
| Цена конверсии, CPA | целое число, ₽ |
| Цена клика, CPC | 1 знак после запятой, ₽ |
| Отказы, % | 1 знак после запятой, % |
| Процент конверсии, CR | 1 знак после запятой, % |
| Кликабельность, CTR | 1 знак после запятой, % |
| Глубина просмотров | 1 знак после запятой |
| Клики, Конверсии, Показы | целое число |

---

## Определение типа кампании

Тип определяется автоматически по названию кампании (приведённому к нижнему регистру).
Приоритет правил сверху вниз:

| Приоритет | Тип | Условие |
|-----------|-----|---------|
| 1 | **Динамическая** | содержит `динамическ` |
| 2 | **РСЯ** | содержит `рся` |
| 3 | **Смарт-баннеры** | содержит `смарт` **И** `баннер` |
| 4 | **БНП** | содержит `поиск` **И** `баннер`, **ИЛИ** начинается с `бнп ` |
| 5 | **Товарная** | содержит `товарн` |
| 6 | **Мастер кампаний** | содержит `мастер` **И** `кампан` (не `товарн`), **ИЛИ** начинается с `МК ` |
| 7 | **Поиск** | содержит `поиск`, не содержит `баннер` и `динамическ` |
| — | **Не определен** | ничего не совпало |

## Требования

- Python 3.9+
- Зависимости указаны в `requirements.txt`

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта.
2. Создайте и активируйте виртуальное окружение:
   ```shell
   # Windows
   python -m venv venv
   venv\Scripts\activate

   # macOS / Linux
   python3 -m venv venv
   source venv/bin/activate
   ```
3. Установите зависимости:
   ```shell
   pip install -r requirements.txt
   ```
4. Создайте папки `data/` и `out/` если их нет.
