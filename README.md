## Анализ отчётов Яндекс.Директ из Excel
Скрипт на Python для автоматического анализа Excel‑отчётов из Яндекс.Директ: считает ключевые метрики (CPC, CPA, конверсия), строит сводки по кампаниям и основным срезам, выводит текстовый отчёт в терминал и формирует Excel‑файл с результатами.

## Возможности
- Чтение Excel‑отчёта из папки `data/` или по указанному пути.
- Автоматическое определение и нормализация колонок (расход, клики, конверсии и т.д.).
- Расчёт метрик:
  - Расход, клики, конверсии.
  - CPC (цена клика), CPA (цена конверсии).
  - CR (конверсия из клика).
- Анализ по основным срезам:
  - Кампания.
  - Устройство.
  - Пол.
  - Возраст.
  - Тип условия показа.
  - Площадки.
- Вывод краткого анализа и рекомендаций в терминал.
- Формирование итогового Excel‑файла с несколькими листами:
  - Обзор по аккаунту.
  - Детальный срез по кампаниям.
  - Листы по каждому разрезу (устройства, пол, возраст, площадки и т.д.).
  - Лист с рекомендациями.

## Запуск

### Вариант 1: Настройка в коде (рекомендуется)
1. Откройте файл `yandex_direct_analyzer.py`
2. В самом начале файла найдите блок настроек:
   ```python
   # ==================== НАСТРОЙКИ ====================
   DEFAULT_INPUT_FILE = "data/BD_RSY_MK_BFL_OPORA_01.03-04.04 2026.xlsx"
   DEFAULT_OUTPUT_FILE = "out/analysis_result.xlsx"
   # ===================================================
   ```
3. Измените `DEFAULT_INPUT_FILE` на путь к вашему файлу
4. Измените `DEFAULT_OUTPUT_FILE` на желаемое имя выходного файла
5. Запустите скрипт:
   ```shell
   python yandex_direct_analyzer.py
   ```

**Важно:** Вы можете хранить несколько файлов в папке `data/`, но для анализа нужно указать конкретный файл в настройках.

### Расширенный запуск
Вы можете указать пути к файлам и другие параметры через аргументы командной строки:

- `--input`: Путь к файлу или папке с файлами (по умолчанию: `data`).
- `--output`: Путь к итоговому Excel-файлу (по умолчанию: `out/analysis_{дата_время}.xlsx`).
- `--min-spend`: Минимальный расход для попадания сегмента в рекомендации.
- `--min-clicks`: Минимальное количество кликов для анализа сегмента.

**Пример:**
```shell
python yandex_direct_analyzer.py --input "C:\reports\my_report.xlsx" --output "C:\reports\analysis.xlsx" --min-spend 1000 --min-clicks 50
