Файл с названием "mainstat_..." должен быть обязательно, так как это главная статистика. Если других файлов нет, то скрипт продолжает работу и не ищет эти файлы, а вкладки отчетов, по которым нет данных не формирует в выходном файле отчета Excel.

В скрипте в начале сделать переменную, в которй мы будем писать свой текст перед запуском скрипта. В теесте мы будем писать название сайта и даты, например "Отчет по сайту Опора за период 1 марта - 04 апреля 2026г." Этот текст потом нужно выводить в ячейке A1 на каждой вкладке с отчетами, размер шрифта 18, жирный шрифт.
В ячейке A2 каждой вкладки нужно написать название отчета, которое соответствует названию вкладки и имеет размер шрифта 16, жирный шрифт.

Скрипт ищет файлы в папке /data/ и идентифицирует их по первым двум словам в названии файла, разделители между словами "_", первое слово - это название отчета, второе слово - название сайта, по которому делается анализ.
- вопрос: Как ты думаешь, реализовать автоматический поиск файлов в папке /data/ или же задавать названия файлов в скрипте вручную? 
Далее я планирую в папку /data/ складывать файлы с данными сразу по нескольким сайтам (по разным проектам и аккаунтам директа) и чтобы скрипт анализировал сразу несколько сайтов за один запуск и для каждого сайта формировал отдельный файл с отчетами. Это не силььно усложнит скрипт? Или же лучше запускать анализ для каждого сайта отдельно и прописывать названия файлов исходных данных при каждом запуске?

1б. Файлы буду выгружены из аккаунта Директ и сохранены без изменений. Это отчеты с данными в разных разрезах. Верхнюю часть файлов - несколько строк с текстами скрипт должен игнорировать, скрипт ищет шапку таблицы и строки таблицы и работает только с данными таблицы из каждого файла.
